# Multi-Agent Debate DAG (LangGraph)

## Overview
This project implements a controlled multi-agent debate system using LangGraph.

## Features
- Strict 8-round enforcement
- Alternating agents
- Memory slicing
- Repetition detection
- CLI-based execution
- Persistent logging
- Judge-based evaluation

## Run
```bash
pip install -r requirements.txt
python run_debate.py --seed 42
```
